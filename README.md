The **Intelligent Resume Screening System** is an AI-powered application that automates resume analysis, classification, and ranking. It enables recruiters to efficiently screen large volumes of resumes by extracting text from PDF files (including scanned resumes), classifying them into job categories, and ranking candidates based on their relevance to a given job description using semantic similarity.

The system reduces manual effort, improves accuracy, and supports fair and efficient hiring decisions.
